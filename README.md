# 💰 Finance Tracker App

A modern Finance Tracker built using React + Vite.

This app helps users manage their income and expenses in a simple and clean interface.

---

## 🚀 Features

• Add income & expenses  
• View transaction history  
• Automatic balance calculation  
• React Router navigation  
• Responsive modern UI  
• Fast performance using Vite  

---

## 🛠️ Tech Stack

React  
Vite  
JavaScript  
CSS  
React Router  

---

## 📦 Installation

Clone the repository:

git clone https://github.com/your-username/FinanceTracker.git

Go to project folder:

cd FinanceTracker

Install dependencies:

npm install

Run the app:

npm run dev
